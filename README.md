# AI ChatGPT Clone UI

A futuristic AI chatbot interface inspired by ChatGPT built using HTML, CSS, and JavaScript.

## Features

- Dark Mode
- Light Mode
- AI Robot Image
- Interactive Chat UI
- Responsive Design
- Smooth Animations
- Modern UI Design
- Direct Run Project

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How To Run

1. Download or Clone Repository
2. Open Project Folder
3. Run index.html

No setup required.

## GitHub Pages

Enable GitHub Pages to host the project live.

Settings → Pages → Deploy from Branch → Main
